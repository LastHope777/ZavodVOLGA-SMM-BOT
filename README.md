# 🤖 SMM Manager Bot for "Завод ВОЛГА"

Этот Telegram-бот автоматизирует процесс **планирования и генерации
контента** для соцсетей (VK, Telegram) компании «Завод ВОЛГА».\
Он использует **OpenAI API / OpenRouter** для генерации расписания и
текстов постов на основе стиля компании.

------------------------------------------------------------------------

## 🚀 Основные возможности

-   📂 Загрузка предыдущих постов (.txt) для обучения стиля
-   🧠 Автоматическая генерация расписания постов на месяц
-   📝 Редактирование расписания перед утверждением
-   ✨ Генерация текстов постов по утверждённому расписанию
-   💾 Сохранение всех данных (постов, расписаний) в SQLite
-   🧰 Полностью автономный процесс --- от планирования до готовых
    текстов

------------------------------------------------------------------------

## ⚙️ Технологии

-   **Python 3.11+**
-   **Aiogram 3.4+** – Telegram Bot Framework
-   **OpenAI / OpenRouter API** – генерация текста
-   **SQLite3** – база данных
-   **Logging** – отслеживание ошибок и действий

## 📦 Установка

### 1️⃣ Клонирование проекта

``` bash
git clone https://github.com/LastHope777/ZavodVOLGA-SMM-BOT.git
cd ZavodVOLGA-SMM-BOT
```

------------------------------------------------------------------------

### 2️⃣ Создание виртуального окружения (рекомендуется)

``` bash
python -m venv venv
```

Активировать:

**Windows:**

``` bash
venv\Scripts\activate
```

**Mac/Linux:**

``` bash
source venv/bin/activate
```

------------------------------------------------------------------------

### 3️⃣ Установка зависимостей

``` bash
pip install -r requirements.txt
```

------------------------------------------------------------------------

## 🔑 Настройка окружения

Создай файл `.env` (или задай переменные среды вручную):

``` env
TELEGRAM_BOT_TOKEN=твой_токен_бота
OPENAI_API_KEY=твой_openai_или_openrouter_api_ключ
OPENAI_MODEL=gpt-4o-mini
```

> 💡 Можно использовать OpenRouter (https://openrouter.ai/) ---
> поддерживает бесплатные лимиты и альтернативные модели.

------------------------------------------------------------------------

## ▶️ Запуск бота

``` bash
python main.py
```

После запуска бот появится онлайн в Telegram.\
Используй команды:

-   `/start` – приветствие и инструкции\
-   `/upload_posts` – загрузка прошлых постов (.txt)\
-   `/generate_schedule` – генерация расписания постов

------------------------------------------------------------------------

## 🗂️ Формат загружаемых постов

Файл `.txt` должен содержать тексты прошлых постов, разделённые пустыми
строками.\
Пример:

    🚰 Автоматические насосные станции Завода ВОЛГА
    Завод ВОЛГА производит современные насосные станции полного цикла...

    🏭 Завод ВОЛГА — надёжный партнёр
    Предприятие работает с 2014 года, реализуя инженерные решения для тепловых пунктов...

## 📋 Пример работы

1.  Отправь `/upload_posts` и загрузи .txt с предыдущими публикациями.\
2.  Используй `/generate_schedule` – бот создаст расписание на
    следующий месяц.\
3.  Утверди расписание ✅ или отредактируй ✏️.\
4.  После утверждения бот создаст тексты всех постов и пришлёт их тебе в
    Telegram.

## 🛠️ Возможные ошибки

| Ошибка | Причина | Решение |
|--------|----------|---------|
| `insufficient_quota` | Нет кредитов на OpenAI | Используй OpenRouter API или пополни баланс |
| `ИИ вернул непонятный формат` | Модель вернула невалидный JSON | Перезапусти генерацию или уточни промпт |
| `Ошибка OpenAI при генерации` | Неверный ключ или таймаут запроса | Проверь `OPENAI_API_KEY` и интернет-соединение |
